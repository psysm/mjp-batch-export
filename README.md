# mjp-batch-export
Mein Justizpostfach (MJP) Batch Export all messages including metadata ("Eingangsbestätigung" und "Prüfvermerk")

1. Login to MJP using Chrome
2. Allow popups and multiple downloads in Chrome.
3. Decrypt messages using your key and password.
4. Open Chrome Developer Tools Console and allow pasting in code.
5. Paste in content of script.js and press enter.

Run on your own risk. Script runs locally. In my test it took about 1 hour to export 500+ messages.
